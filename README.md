# Virtual Jewellery Try-On

Real-time virtual jewellery try-on application using YOLOv8, MediaPipe and OpenCV.

## Features
- Face Detection
- Hand Tracking
- Necklace Placement
- Earring Placement
- Ring Placement
- Real-time Webcam Support

## Tech Stack
- Python
- OpenCV
- MediaPipe
- YOLOv8
- NumPy

## Installation

pip install -r requirements.txt

## Run

python main.py
